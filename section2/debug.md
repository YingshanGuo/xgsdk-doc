# 模拟接入
----

模拟接入是在正式接入测试之前进行的接入调测，验证基本接入参数信息，提高正式接入的成功率。  
模拟接入需要配合客户端的调试工具和西瓜web控制台的调测信息进行，本章节主要讲解模拟接入客户端工具的使用。  
客户端工具目前支持Cocos2d-x和Unity3d两个引擎。请游戏选择与自己相对应的引擎模拟接入文档进行参考，文档详细的描述了模拟接入环境、接入步骤、参考代码和接入时的注意事项，能够有效地提高模拟接入效率。
