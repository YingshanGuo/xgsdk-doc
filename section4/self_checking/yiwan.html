文档中所有接口均为必接接口，包括生命周期接口,SDK对接完成后请使用自检APK工具进行测试，通过后将游戏包与自测文档一并提交。
分类 测试点 测试点详细描述
版本信息检查 游戏版本号 version code 必须为整数，并且新版本的version code要比旧版本的大
versionCode、versionName非0非空，也不能与前版本相同
游戏包名 Androidmanifest.xml中package字段需要区别于其他渠道，游戏包名需以“.ewan”结尾，如：com.tips.fbzs.ewan
游戏版本更新，package字段需要与已上架版本一致（区分大小写）
图标检查 游戏icon 游戏的icon图标必须加入益玩logo，要求角标位置不影响icon的主体展现
游戏中配置的所有icon文件均需打上益玩logo，logo位置必须保持一致
接口检查 初始化 游戏必须保证最先调用SuperSDK的初始化接口，在接收到SuperSDK初始化成功后方可进行游戏的其他操作(如登陆、支付等)
游戏版本号必须在Androidmenifest.xml中配置(如:android:versionName="1.0.0")，不能通过其他配置文件读取，否则无法开启更新
调用初始化接口后，正常弹出更新提示，选择"立即更新"正常更新游戏包(测试包)，"取消更新"正常游戏
必须保证益玩的更新优先于CP的更新模块
游戏包必须为正式环境包，初始化接口中 DebugMode=1
游戏公告不能在益玩的初始化接口调用前显示，不能遮挡益玩更新提示框
初始化过程中不能出现闪屏、黑屏现象
必须处理初始化失败(INIT_FAIL)的回调消息
释放资源 游戏界面按下手机返回键（退出游戏时），弹出退出提示框并且能正常退出游戏
游戏内的所有退出按钮，点击后都需要调用SuperSDK的exit释放资源接口与logout退出接口，在回调消息中做相关退出处理
益玩账号系统 正常调出登录界面，点击"模拟登陆"按钮正常登陆游戏; 点击"取消登陆"、"登陆失败"正常返回登陆界面;
必须处理login接口中登录成功(LOGIN_SUCCESS)、登录失败(LOGIN_FAIL)、取消登录(LOGIN_CANCEL)、切换账号(SWITCH_ACCOUNT的)回调消息
益玩支付 确定游戏是使用固定金额还是可选金额充值方式
确定好游戏的游戏币名称
确定好人民币跟游戏币的兑换比例
确定好游戏接入SDK服务器的充值回调地址是游戏的正式服的地址
支付API接口的 服务器ID字段必须准确
支付API接口的 CustomInfo字段必须传入游戏当前购买的订单号，订单号需只含有数字、字母、下划线，且不能重复
支付金额需在3-5分钟内到账
金额到账后必须能及时刷新，无需重启游戏；若无法自动刷新，需弹框提示，告知玩家如何操作或者添加手动刷新按钮
到账的游戏币数量正确。若有充值活动导致到账的游戏币数量不正确，需在公告或者是商品列表中有相关文字说明
充值额外赠送的东西，例如额外赠送多少元宝，或者是奖品，能正常发放
游戏中商品名称、数量、价格必须和益玩充值界面显示的名称、数量、价格一致（游戏是固定金额充值方式的时候）
游戏订单号要求 订单号需只含有数字、字母、下划线，且不能重复
不含非益玩支付方式 游戏中需不含有任何非益玩的充值方式以及非益玩充值方式的介绍（如活动、公告界面；帮助界面），即使是点击无响应的按钮（必须屏蔽相关按钮）；
API接口调用与检测 文档中所有接口均为必接接口，包括所有生命周期接口
更新检查 游戏升级流程正确 游戏需保证启动游戏后，先进行SuperSDK的初始化处理（益玩的初始化流程里面包括了升级流程处理），弹出更新提示框，并且用户在提示框操作后再进行后续的登陆动作
游戏包更新 点击“立即更新”能够正常下载并且安装游戏包(此处为测试包)，点击“稍后更新”能够正常进行游戏
不含非益玩的客户端更新检查 游戏不含非益玩的客户端检查方式（包括非益玩客户端更新检查的按钮以及非益玩客户端更新检查的链接）
数据采集 益玩数据采集 创建角色的时候，需要调用  SuperSDK的 数据采集接口
玩家角色登陆的时候，也需要调用  SuperSDK的 数据采集接口
客服、外链检查 游戏内显示的内容、调用通讯工具或社交软件功能的默认发送内容需不含非益玩引导 需不含引导用户到其他下载地址下载非益玩渠道的客户端（包括官网）
需不含引导用户使用非益玩支付方式的相关信息
需不含引导用户到其他游戏平台进行好评，给予奖励
需不含引导用户访问非益玩官网、论坛的相关信息
需不含官网、官方论坛等类似访问官方网站按钮，即使是灰状的，无法点击的
需不含益玩用户无法参与的分享活动
其他 敏感词屏蔽 游戏中任意可以输入文本的地方，均需对【钓鱼岛】、【尖阁列岛】等敏感词的发送进行屏蔽
不能出现非益玩的游戏平台SDK和广告SDK 若发现接入其他游戏平台的SDK或者第三方的广告SDK，直接打回
游戏名称命名规则 游戏名称_ewan_游戏当前版本号_游戏文件的提供日期（比如：风暴战神 3.1.5版   fbzs_ewan_v315_20140814.apk）
赌博相关字眼检查 游戏所有UI界面，功能，不能出现这些字眼:炸金花、诈金花、扎金花、梭哈、赢三张、三张牌、港式五张、赌场、赌、六合彩 
游戏安装包的固件要求必须写 游戏安装包的固件要求需配置
即需要设置AndroidManifest.xml文件中的minSdkVersion和targetSdkVersion
