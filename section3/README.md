#西瓜SDK服务端接入指南
----


本章节介绍游戏服务器如何接入西瓜服务器，适用于游戏服务端开发人员。游戏服务端开发人员可以忽略客户端引擎的差异，直接参考本章节的接入文档进行接入，其中账号验证和支付验证为必接接口（接入文档分别对应** [session验证](./session.md) ** 和 **[支付通知](./pay.md)**）。在进行服务端接入之前，建议首先阅读《西瓜SDK流程说明》，可以有效的提高接入效率。
